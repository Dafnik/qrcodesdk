## One builder, focused renderers

QRCodeSDK is organized as a small package family. The core package generates QR matrices and includes renderers that work in any JavaScript runtime. The Node package adds server-side renderers for output that depends on Node APIs.

<CardGrid>
  <Card title="@qrcodesdk/core" icon="seti:typescript">
    Generate QR matrices and render SVG strings or terminal text strings in browser, server, CLI,
    and test environments.
  </Card>
  <Card title="@qrcodesdk/node" icon="seti:nodejs">

  </Card>
</CardGrid>

## Quick start

1. Install the core package.

   ```sh
   pnpm add @qrcodesdk/core
   ```

2. Render your first SVG.

   ```ts
   import {SVGQRCodeRenderer, qrcode} from '@qrcodesdk/core';

   const svg = qrcode('https://qrcodesdk.dev')
     .renderer(SVGQRCodeRenderer({size: 6, margin: 4}))
     .render();
   ```

3. Add Node output when you need PNG files.

   ```sh
   pnpm add @qrcodesdk/node
   ```

## Render where you need it

### SVG

```ts
import {SVGQRCodeRenderer, qrcode} from '@qrcodesdk/core';

const svg = qrcode('https://qrcodesdk.dev')
  .renderer(
    SVGQRCodeRenderer({
      colors: {
        colorLight: '#ffffff',
        colorDark: '#111111',
      },
      ariaLabel: 'Open QRCodeSDK',
    }),
  )
  .render();
```

### Terminal

```ts
import {QRCodeTextRenderer, qrcode} from '@qrcodesdk/core';

const output = qrcode('HELLO WORLD')
  .mode('alphanumeric')
  .renderer(QRCodeTextRenderer({size: 1, margin: 2}))
  .render();

console.log(output);
```

### PNG

```ts
import {writeFileSync} from 'node:fs';

import {qrcode} from '@qrcodesdk/core';
import {PNGQRCodeRenderer} from '@qrcodesdk/node';

const png = qrcode('https://qrcodesdk.dev')
  .renderer(PNGQRCodeRenderer({size: 8, margin: 4}))
  .render();

writeFileSync('qr.png', png);
```
