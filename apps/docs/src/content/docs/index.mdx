QRCodeSDK keeps QR code generation separate from presentation. Start with `qrcode()`, then output SVG, PNG, Canvas, image elements, terminal text, matrix data, or your own custom renderer.

## Install and render SVG

1. Install `@qrcodesdk/core`.

   <PackageManagerTabs packages={['@qrcodesdk/core']} mode="add" />

2. Render your first QR code.

   ```ts
   import {QRCodeSVGRenderer, qrcode} from '@qrcodesdk/core';

   const svg = qrcode('https://qrcodesdk.dev').render(QRCodeSVGRenderer());
   ```

SVG is the best default for most user-facing QR codes because it stays sharp at any size and works in browsers, servers, emails and static assets.

## Customize the QR code

Most renderers use the same styling options.

```ts
import {QRCodeSVGRenderer, qrcode} from '@qrcodesdk/core';

const svg = qrcode('https://qrcodesdk.dev')
  .errorCorrection('H')
  .render(
    QRCodeSVGRenderer({
      size: 8,
      margin: 4,
      colors: {
        colorDark: '#111827',
        colorLight: '#ffffff',
      },
      ariaLabel: 'Scan to open qrcodesdk.dev',
    }),
  );
```
